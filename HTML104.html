Key Notes in HTML

Heading Tag: The heading tag consists of 6 heading tags that is from <h1>-<h6> tags. The size of each heading reduces as the number of the heading tag increases. That means that the largest size for heading would be <h1> while the smallest size for heading will be <h6>. Heading is generally used for what the name means, to assign heading to a group of texts.

Paragraph Tag: The paragraph tag is a tag that has <p> as it's tag. The paragraph tag is not like the heading tag in the sense that there is just one "p" that makes up the tag. As the name suggests the paragraph tag is used to add paragraphs to the page.  

Anchor Tag: Have you been to a website and you see something like "click here to visit this page" and then the here is kind of highlighted and when clicked, you get sent to another page? That's probably the actions of an anchor tag. The anchor tag generally represented by <a> is used to provide a link to another page, media or document stored on your local server or an external server. The anchor tag is used together with an attribute called "href". "href" allows you to point to these files so if I wanted to anchor a page from my current page while using "here" as the anchor, it would be like: click <a href="page.html">here</a> to visit my page. When you execute the code, the here will be highlighted and when clicked it would open the page. By default, when you click on an anchor link, it opens whatever is anchored on the current page tab, we can change this behavior by using a target attribute. This will create a new tab on the browser while keeping the previous tab intact. To do this we do click <a href="page.html" target="_blank">here</a> to visit my page. If page.html exists on the server, it will open on a new page.

Ordered list Tag: Ordered list tag otherwise called numeration tag is a tag that is used to create a numerated list like 1. ,2. , 3. etc. To make an ordered list you actually need two tags; The ordered list tag itself <ol> and then the list tag represented by <li>. The <li> element is actually nested within the <ol> element. So the next time you see a website that has numerated items, you know what's going on.

Unordered list Tag: Unordered list tag otherwise called bullet tag is a tag that's used to create a list of bullet listed items. You may have the bullets as small circles (default) or even squares and with a bit of CSS, you can make the bullets images of your choice! The unordered list tag also comprises of two tags; The unordered list tag itself represented by <ul> and then the list element <li> ,which is also nested in the unordered list tag. It is important to note that if you just use the <li> element without nesting it in either an ordered or unordered list tag, the items will carry bullets(unordered list) by default.

Semantic Tags: One thing you need to keep in mind is SEO of your website. SEO ranking is simply how Google and other search engines rank websites. What this implies is that if a website has good SEO, they appear on top search term results for users and one of the best ways to boost SEO is using Semantic elements

By default using the website builder already contains a couple of semantic elements and we shall be talking about them. Note that semantic elements don't exactly add special actions to the other elements nested within them. They just help in organizing code by making it readable and understandable and boosting SEO.

  • Header Element: The header element represented by <header> is simply used to contain the top parts of the website page which could contain the navigation items (hamburger menu) and aside element. 
  • Navigation Element: The navigation element represented by <nav> is usually inside the header element and it's major purpose is to house the menu bar and items otherwise called hamburger. By default, the only type of menu supported by the web builder is Hamburger menu
  • Section Element: The section element represented by <section> is a widely used element in the website builder. The section elements specifies where a new section of the webpage starts and ends!
  • Aside Element: The aside element represented by <aside> is typically used to suggest that certain elements will appear from one side of the screen. This is best for building drawer menu. This menu type is however not applicable in the web builder
  • Main Element: The main element represented by <main> is usually used immediately after the body element. It is used to signify the main part of the webpage.
  • Footer Element: The footer element represented by <footer> is usually used to show the end credits of a website as well as important links and information. The footer is generally located at the bottom of the page.
