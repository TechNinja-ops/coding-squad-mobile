The HTML Structure

In HTML, it's a CRIME to write naked texts. What are naked texts? Everything you're currently seeing on your screen is called naked texts. If you look at the source code of this page you'll see that the texts are enclosed in something called tags which are generally angle brackets.

In HTML, texts MUST be enclosed in angle brackets <> or </> . The angle brackets are of two types <> is for starting and </> is for ending. What do we mean? To understand what the angle brackets is meant for, we need to visit a concept called HTML elements. Now a HTML element will vary depending on what type of content you're making. Let's say we have a text on the website that's meant to say: Hello Student, welcome to Coding Squad". This should be a paragraph so we'll need something called the paragraph tag.


The line 1 is considered naked texts and though it will be shown on the browser exactly how it is, this is a bad way to treat HTML elements. Line 4 however is properly put with the angle brackets enclosing the letter P. If you notice, the first angle bracket and the last angle brackets are somewhat different. You should have a better understanding what angle brackets are. We have compiled a cheat sheet for HTML tags where you can better understand what element to use aside from "p". Play around with it!  When we say HTML tag we mean just a simple expression in HTML such as <p></p> in essence, when there's no content between the starting and ending angle brackets.
