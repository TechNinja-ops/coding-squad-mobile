CSS Structure

Writing CSS is a little different from writing HTML, you see CSS isn't a programming language; It's a rule set basically defining how an element should look on the webpage. Remember the example of a dress we used to understand why CSS is necessary? The identifier given to the dress is more like ids and classes but in CSS you can style other things including elements! In CSS, HTML elements are called CSS selectors and how it works is that you target the element you want to apply styles to by simply typing the element's name followed by a curly brace '{}'. By default, Vscode will automatically close the curly brace for you and then between the curly braces you type out the CSS rules followed by a semi colon to indicate the end of a rule.
