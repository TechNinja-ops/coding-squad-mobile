What is HTML and CSS

HTML stands for Hyper Text Markup Language. It's arguably the easiest thing to learn and as the name implies, HTML isn't a programming language, it's a markup language which ideally tells a web page where things are meant to be. The totality of HTML includes adding texts, images, audio and video. Aside that there's not any complex thing here.

CSS stands for Cascading Style Sheets. It is what actually gives the look and appearance of the HTML content on your webpage. Again, CSS isn't a programming language, it's pretty much just rules on how different elements are meant to look like on the page.

Making a standard website actually requires three things and aside the aforementioned two, there's another one called JavaScript. This is arguably the most in-demand and broad languages to get your paws on. We would not be teaching anything on JavaScript here as there wouldn't be any need to touch the JavaScript code. HTML, CSS and JavaScript can be likened to the human body. Think of HTML as the skeleton of the human body, CSS is like the skin which also includes your height, your dimples, your physique etc. And JavaScript is like the organs in your body. If you see a website written in just HTML, it's like imagining a skeleton in front of you, scary right? Then if you have a website written in just HTML and beautiful CSS, imagine a human with just skeleton and nice physique but with no internal organs, the website may look normal but it's very incomplete, just a carcass right?
